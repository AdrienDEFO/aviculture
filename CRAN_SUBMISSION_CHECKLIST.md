## Common CRAN Rejections & Fixes

| Issue | Fix |
|-------|-----|
| "Title is too long" | Shorten to < 65 chars |
| "Imports not used" | Remove unused packages from DESCRIPTION |
| "Examples take too long" | Wrap in \dontrun{} or \donttest{} |
| "Non-ASCII characters" | Use UTF-8 encoding |
| "Hardcoded paths" | Use system.file() instead |
| "No example output" | Add proper @examples section |
