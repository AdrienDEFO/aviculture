## 📚 USAGE EXAMPLES

After installation:

```r
library(aviculture)

# Load model
model <- load_model()

# Basic predictions
mass_30 <- predict_mass(30, model)
volume_30 <- predict_volume(30, model)
density_30 <- predict_density(30, model)

# Growth simulation
trajectory <- simulate_growth(start_age = 1, end_age = 60, model = model)
plot(trajectory$age, trajectory$mass_ref)

# Tissue analysis
composition <- get_tissue_composition(30, model)
cat("Muscle at day 30:", composition$muscle, "%\n")

# Data validation
validation <- validate_measurements(30, 2.2, 2.1, model)
print(validation$messages)

# Model info
get_model_info(model, verbose = TRUE)
```

## ❌ POTENTIAL ISSUES & SOLUTIONS

### Issue: "avimodel.pkl not found"
**Solution:** Ensure avimodel.pkl is in `inst/extdata/`:
```powershell
copy d:\projet\Avipro\avimodel.pkl d:\projet\Avipro\aviculture\inst\extdata\
```

### Issue: Roxygen documentation not generated
**Solution:** Make sure Roxygen comments are correct (they are!), then run:
```r
roxygen2::roxygenise()
```

### Issue: Package check shows "Hidden objects"
**Solution:** This is normal for internal functions. Can be ignored for CRAN.

### Issue: Python pickle module not found
**Solution:** Make sure reticulate is configured:
```r
reticulate::py_config()
reticulate::use_python("C:/Users/..../Python39/python.exe")
```
